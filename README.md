# dr_dadata_drupal_11
Модуль для тестирования сервиса Dadata для Drupal 11.

После установки модуля в разделе Конфигурация (вкладка Веб-службы) появляется пункт меню Dadata API.

В разделе Dadata API - 2 поля API Token и Secret Key.
Эти данные берем из личного кабинета Dadata.

Переходим на страницу site.local/dadata
и в поле ввода ИНН - прописываем любой ИНН - выводится информация об организации.

Польза от модуля сомнительна, больше как практика по созданию модуля для Drupal 11.
